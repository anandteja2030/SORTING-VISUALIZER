# 🚀 Sorting Visualizer

An interactive **Sorting Visualizer** built using **HTML, CSS, and JavaScript** that helps users understand how popular sorting algorithms work through real-time animations and visual representations.

This project provides an engaging way to learn sorting techniques by displaying each array element as a vertical bar and animating every comparison, swap, and sorting step. It also includes performance statistics, customizable arrays, and step-by-step execution for a better learning experience.

---

# ✨ Features

- 🎯 Visual representation of sorting algorithms
- 📊 Supports multiple sorting algorithms
- 🎨 Modern and responsive user interface
- 🎲 Generate random arrays instantly
- ✍️ Create custom arrays with your own elements
- 📏 Customize array size
- ⚡ Adjustable animation speed
- 🪜 Step-by-Step execution mode
- ▶️ Start sorting with smooth animations
- ⏹️ Stop sorting at any time
- 📈 Live comparison counter
- 🔄 Live swap/write counter
- ⏱️ Real-time execution timer
- 🎨 Color-coded visualization for comparisons, swaps, and sorted elements
- ⌨️ Keyboard shortcuts for quick interaction
- 📱 Responsive design for different screen sizes

# 📊 Algorithm Complexity

| Algorithm | Best | Average | Worst | Space |
|------------|------|---------|--------|--------|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) |

# 🖥️ Project Workflow

### 1️⃣ Generate an Array

Users can:

- Generate a random array
- Enter a custom array
- Specify their preferred array size

---

### 2️⃣ Choose a Sorting Algorithm

Select any one of the available algorithms:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

---

### 3️⃣ Adjust Animation Speed

Move the speed slider to control how fast or slow the visualization runs, making it easier to observe every operation.

---

### 4️⃣ Start Visualization

Click **Start Sorting** to begin the visualization.

During execution, the application displays:

- Current algorithm status
- Number of comparisons
- Number of swaps/writes
- Execution time

---

### 5️⃣ Step Mode

Enable **Step Mode** to execute the algorithm one operation at a time. This feature is especially useful for beginners who want to understand every comparison and swap in detail.

---

# 📊 Live Statistics

The application tracks:

- 📈 Comparisons
- 🔄 Swaps/Writes
- ⏱️ Execution Time
- 📌 Current Algorithm Status

# ⌨️ Keyboard Shortcuts

| Key | Action |
|------|--------|
| **Enter** | Start Sorting |
| **S** | Enable Step Mode |
| **Space** | Execute Next Step |
| **R** | Generate New Array |
